# 航空航天模型扩展

## 1. 目标

本文档用于补齐 `knowledge_base/docs/` 中与航空航天建模相关的领域知识、标签体系、术语词典、参数词典与路由提示，重点服务以下场景：

- 运载火箭与发射动力学
- 导弹 / 拦截弹二维飞行轨迹
- 卫星 / 航天器二体轨道建模

本文档既服务知识检索，也服务“无匹配先澄清 / 拒绝误配”的领域判断。

---

## 2. 领域边界

### 2.1 本域核心问题

航空航天域优先处理以下类型的请求：

- 发射、升空、推力、燃料消耗、垂直起飞
- 导弹、拦截、飞行轨迹、发射角、弹道、空气阻力
- 卫星、轨道、二体问题、轨道高度、近地轨道、初始速度
- 航天器、轨道机动、空间飞行、重力参数

### 2.2 与相近领域的边界

容易与航空航天域混淆的请求包括：

- “水下发射、潜艇、鱼雷、浮力、水阻” → 应优先归到军工装备域的水下发射
- “雷达跟踪、目标航迹、观测噪声、卡尔曼滤波” → 应优先归到军工装备域或战场态势域的跟踪估计
- “红蓝对抗、兵力消耗、战损、态势评估” → 应优先归到战场态势域

如果用户只说“发射模型”，但没有说明对象是火箭、导弹还是鱼雷，必须先澄清对象与介质，不能直接误配。

## 5. 术语词典

### 5.1 推进与发射术语

| 术语 | 英文 / 别名 | 含义 |
| --- | --- | --- |
| 推力 | thrust | 发动机或推进装置沿主轴方向提供的力 |
| 推重比 | thrust-to-weight ratio | 推力与重力的比值，决定起飞能力 |
| 燃料质量 | fuel mass | 可消耗推进剂质量 |
| 燃烧速率 | burn rate | 单位时间消耗的推进剂质量 |
| 发射质量 | initial mass / mass0 | 起飞时总质量 |
| 发射角 | launch angle | 导弹或飞行器初始方向角 |
| 初始速度 | init speed / v0 | 初始时刻速度大小 |
| 空气密度 | air density | 用于估计空气阻力 |
| 阻力系数 | drag coefficient / Cd | 反映气动阻力强弱 |
| 参考面积 | reference area / area | 阻力与升力估计常用面积参数 |

### 5.2 轨迹与飞行动力学术语

| 术语 | 英文 / 别名 | 含义 |
| --- | --- | --- |
| 飞行轨迹 | trajectory | 飞行器随时间变化的位置曲线 |
| 弹道 | ballistic trajectory | 受初始速度、推力与重力共同影响的轨迹 |
| 平面轨迹 | planar trajectory | 在二维平面内近似建模的轨迹 |
| 阻力 | drag | 与速度方向相反的流体阻力 |
| 重力 | gravity | 地球或中心天体施加的引力加速度 |
| 推进段 | boost phase | 导弹 / 火箭有持续推力的飞行阶段 |
| 末段 | terminal phase | 靠近目标或再入前后的飞行阶段 |
| 射程 | range | 水平或总路径飞行距离 |

### 5.3 轨道力学术语

| 术语 | 英文 / 别名 | 含义 |
| --- | --- | --- |
| 二体问题 | two-body problem | 只考虑中心天体与飞行器之间引力的模型 |
| 轨道高度 | altitude | 距离中心天体表面的高度 |
| 轨道速度 | orbital speed | 保持给定轨道所需的速度 |
| 引力参数 | mu / gravitational parameter | 天体质量与引力常数合成参数 |
| 近地点 | perigee | 轨道上最接近地球的位置 |
| 远地点 | apogee | 轨道上距离地球最远的位置 |
| 轨道周期 | orbital period | 绕中心天体一周所需时间 |
| 近地轨道 | LEO | 近地低轨道场景 |

## 7. 典型建模意图与路由提示

### 7.1 应优先路由到 `rocket_launch_1d`

- 构建一个一维火箭垂直发射模型
- 生成运载火箭起飞阶段 MATLAB 脚本，考虑推力、阻力和质量递减
- 做一个升空模型，给定初始质量、燃料质量和燃烧速率

### 7.2 应优先路由到 `missile_flight_2d`

- 构建导弹二维飞行轨迹模型
- 考虑导弹发射角、初速、推力和空气阻力
- 做一个拦截弹飞行弹道仿真

### 7.3 应优先路由到 `satellite_orbit_2body`

- 构建卫星二体轨道模型
- 生成近地轨道航天器二维轨道脚本
- 给定轨道高度和初速度，仿真卫星轨道

### 7.4 应先澄清的表达

- 发射模型
- 弹道模型
- 飞行器轨迹模型
- 航天模型
- 空天飞行模型

以上表达如果没有明确对象、介质或任务层次，应先澄清：

- 是火箭、导弹还是卫星？
- 是发射段、飞行段还是轨道段？
- 是空气环境还是太空轨道环境？

## 8. 反误配规则

### 8.1 不应把这些请求误配为火箭发射

- 潜艇水下发射鱼雷
- 水下阻力与浮力问题
- 声呐跟踪目标

### 8.2 不应把这些请求误配为导弹轨迹

- 卫星轨道传播
- 红蓝兵力消耗
- 雷达滤波跟踪

### 8.3 不应把这些请求误配为轨道模型

- 有持续推力且关注发射角的近程弹道问题
- 火箭起飞阶段动力学
- 防空导弹拦截轨迹

## 10. 使用建议

对于航空航天域请求，优先识别以下三类要素：

1. 对象：火箭、导弹、卫星、航天器
2. 环境：大气层内、近地轨道、空间飞行
3. 目标：发射、飞行轨迹、轨道传播

只有在这三个层面同时比较清晰时，才应直接进入代码生成；否则应先澄清。
